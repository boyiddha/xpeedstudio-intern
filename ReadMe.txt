					
				Installation Process:


	•Read the ReadMe.txt file carefully! Here you’ll get the instructions to run the project.
	
	•Install XAMPP into you PC. Open  "XAMPP control panel"  and start Apache and MySQL.

	•Unzip the file

	•Copy the  "xpeedstudio" folder and paste it into this location => \xampp\htdocs

	•Open phpMyAdmin and create a Database namely  "registration" . N.B. Database name must be  "registration".

	•Open the Database Folder, here exist a SQL file. Import this  "registration.sql"  file into the registration DataBase.
 
	•Now type   "localhost/xpeedstudio"  into your browser.
